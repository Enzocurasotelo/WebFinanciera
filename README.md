# EncusFinance

**EncusFinance** es un proyecto que integra el desarrollo de software con una identidad de marca profesional. Contiene una plataforma web financiera y su manual de identidad corporativa.

## Tecnologías y Herramientas
* **Frontend:** HTML, CSS y JavaScript.
* **Diseño de Marca:** Manual corporativo (PPTX incluido).
* **Conceptos:** Interfaz minimalista, estética moderna y experiencia de usuario (UX).

## Estructura del Proyecto
* `/EncusFinance`: Código fuente de la web.
* `MANUAL CORPORATIVO ENCUSFINANCE.pptx`: Guía de estilo y visión de la marca.

## Objetivo
Desarrollar una presencia digital coherente para una entidad financiera, aplicando estrategias de marketing digital y diseño funcional.
